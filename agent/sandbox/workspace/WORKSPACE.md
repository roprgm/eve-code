# Workspace

A minimal Vite app with dependencies already installed. It is a head start, not a
constraint: grow it, replace it, or turn this workspace into something else entirely.

- `index.html` + `src/main.js` + `src/style.css` — the entire app
- `npm run dev` — dev server, already configured for this sandbox's public URL
- `npm run build` — production build into `dist/`
